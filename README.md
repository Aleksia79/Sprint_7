# Sprint_7
- Проект автоматизации тестирования API онлайн-сервиса заказа самокатов "Яндекс.Самокат"

## Требования
- Для запуска тестов должны быть установлены пакеты pytest и requests
- для получения отчета должен быть установлен пакет allure_pytest 
- Запуск всех тестов выполняется командой pytest -v
- URL и endpoints в config.py 
- тестовые данные для регистрации курьера и оформления заказа в data.py
- allure-отчет - в папке allure_results
- allure-отчет в формате веб-страницы формируется командой allure serve allure_results
- список внешних зависимостей в requirements.txt
